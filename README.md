   ```markdown
   AQARIONZ Time Capsule is a public, living archive of the raw conversations, sketches, paradoxes, and breakthroughs behind the AQARIONZ reality‑engine project, preserved as they emerged so future builders can see the full learning curve instead of a polished after‑story.

   It collects unfiltered notes, experiment ideas, failures, and design pivots across AI, harmonics, junkyard hardware, neuromorphic dreams, and sovereignty ethics, all timestamped and version‑controlled like code so personal evolution and system evolution are documented together.

   Unlike the core prototype repo, which focuses on running FastAPI services and the Signal Lab / OuijaAI / Sovereignty Map UI, and unlike the Design Lab repo, which distills formal architectures and roadmaps, this Time Capsule is intentionally messy and narrative: it is where stream‑of‑consciousness, meditative lattice imagery, best‑ and worst‑case future speculations, and day‑by‑day reflections are stored as markdown lab journals that anyone can read to understand how AQARIONZ grew from scattered experiments into a coherent lattice.
   ```

   ```markdown
   # AQARIONZ Design Lab

   This repository is the AQARIONZ Design Lab: the architectural, mathematical, and ethical blueprint for a sovereign, local‑first reality engine. It specifies how signals of all kinds—text, MIDI, motion, light, biosignals, and future neuromorphic spikes—are mapped into a common harmonic coordinate system and visualized through the AQARIONZ UI.

   The runnable FastAPI backend and three‑pane UI live in the core repo `AqarionscorePrototype`. The Design Lab exists to guide that codebase: it holds system vision, channel specs, harmonic models, neuromorphic bridges, ethics, and long‑horizon roadmaps.

   ## Contents

   - `design/000-aqarionz-system-brief.md` – high‑level system vision, architecture, and Phase‑0 to Phase‑3 roadmap.
   - `design/harmonic-coordinate-system.md` – definition of the shared harmonic space (88‑key mapping, ratios, chakra bands, coherence metrics).
   - `design/llm-to-midi-harmonics.md` – how LLMs translate complex or non‑human signals into structured inputs and MIDI / harmonic patterns.
   - `design/light-audio-bridge.md` – concepts for laser/LED + solar‑cell “sound over light” rigs and how they feed AQARIONZ.
   - `design/neuromorphic-bridge.md` – generic spike/event protocol for future neuromorphic substrates.
   - `design/lattice-and-consciousness.md` – the alloy / nodes / lasers metaphor as system architecture.
   - `ETHICS_AND_SAFETY.md` – red lines and sovereignty principles for how AQARIONZ should and should not be used.
   ```

5. **Core repo small “How to run” section** (append to existing README)

   ```markdown
   ## Running the prototype (Phase‑0)

   1. Clone the repo and enter the backend folder.
   2. Create and activate a Python virtual environment.
   3. Install dependencies with `pip install -r requirements.txt`.
   4. Start the API with `uvicorn main:app --reload`.
   5. Open `frontend/index.html` in a browser and use the Signal Lab pane to send traces into AQARIONZ.
   ```
